# 承億搬運客戶資料回填問卷

用途：這份問題是為了補齊網站上的 `{{PLACEHOLDER}}`，讓官網可以正式上線，並強化 Google 搜尋、在地 SEO、AI Search / GEO 與 LocalBusiness 結構化資料。

## P0 必問：不上線前最好先確認

### 1. 正式網站網址

- 官網正式網域會是什麼？
- 是否已經決定使用 `https://` 的正式網址？
- 是否會使用 `www`？例如 `https://www.example.com` 或 `https://example.com`

對應欄位：

- `{{DOMAIN_PLACEHOLDER}}`

### 2. 商家完整地址

- 承億搬運是否有可公開顯示的營業地址？
- 如果可以公開，完整地址是什麼？
- 如果不想公開完整地址，是否可公開到「縣市 + 行政區」？

對應欄位：

- `{{FULL_ADDRESS_PLACEHOLDER}}`

### 3. 主要服務區域

- 主要服務區域是哪裡？例如：高雄市、鳳山區、左營區、三民區、大高雄地區。
- 是否有跨縣市服務？
- 是否有全台回頭車、長途搬運或外縣市搬家？
- 哪些區域是最想主打的 SEO 關鍵字？

建議請客戶提供：

- 主要區域 1 個
- 常服務行政區 5-10 個
- 可服務但非主力的區域

對應欄位：

- `{{SERVICE_AREA}}`
- `{{SERVICE_AREA_LIST_PLACEHOLDER}}`

### 4. 營業時間與夜間服務

- 平常營業時間是幾點到幾點？
- 是否 24 小時接單？
- 夜間搬家是否需要提前預約？
- 夜間搬家是否加價？如果會，加價規則是什麼？
- 國定假日或週末是否服務？

對應欄位：

- `{{HOURS_PLACEHOLDER}}`

## P1 重要：影響地圖、信任與轉換

### 5. Google 地圖資料

- 是否已有 Google 商家檔案？
- Google Maps 店家連結是什麼？
- Google Maps 導航連結是什麼？
- 是否知道 Google Place ID？
- 是否知道店家座標 lat / lng？

對應欄位：

- `{{GMAPS_DIRECTIONS_URL}}`
- `{{PLACE_ID_PLACEHOLDER}}`
- `{{LAT}}`
- `{{LNG}}`

### 6. 計價與價格區間

- 搬家通常怎麼計價？以車計價、包價、樓層、距離、物品數量，還是現場估價？
- 是否有基本起跳價？
- 自助搬家、家庭搬家、夜間搬運是否有不同收費方式？
- 無電梯樓層是否加價？每層如何計算？
- 大型家具、鋼琴、魚缸、精密儀器是否另計？
- 是否可提供一個公開的價格區間，例如 `NT$1,500 起` 或 `依現場估價`？

對應欄位：

- `{{PRICE_RANGE_PLACEHOLDER}}`
- FAQ 計價內容
- Services 服務頁 CTA 文案

### 7. 包材與設備細節

目前網站已寫入：

- 專業紙箱：50 x 35 x 35cm
- 大型打包袋：50 x 65cm
- 升降尾門貨車：長 285 x 寬 169 x 高 115cm

請確認：

- 以上尺寸是否正確？
- 紙箱與打包袋是免費提供、借用、販售，還是依方案提供？
- 是否提供氣泡布、保護膜、棉被、防撞墊、膠帶？
- 貨車是否都有升降尾門？
- 是否有其他車型或更大車型？

對應內容：

- 首頁裝備展示
- Services 服務說明
- FAQ 包材問題
- LocalBusiness trust signals

### 8. 服務項目確認

目前網站有五大服務：

- 家庭精緻搬家
- 自助搬家
- 夜間運輸
- 精密儀器護送
- 專業打包

請確認：

- 這五項是否都確實提供？
- 哪一項最想主打？
- 是否有不想公開宣傳的服務？
- 是否提供公司搬遷、學生搬家、小套房搬家、倉儲搬運？
- 大型廢棄物是否協助清運？是否合法代收或只是協助搬到指定地點？

對應內容：

- `/services`
- 首頁服務卡片
- FAQ
- Blog 文章內部連結方向

## P2 加分：補強品牌信任與 AI Search

### 9. 社群連結

- Threads 連結是什麼？
- Facebook 粉專連結是什麼？
- Instagram 連結是什麼？
- 是否有 TikTok、YouTube 或其他平台？

對應欄位：

- `{{THREADS_URL_PLACEHOLDER}}`
- `{{FB_URL_PLACEHOLDER}}`
- `{{IG_URL_PLACEHOLDER}}`

### 10. Email 與其他聯絡方式

- 是否要公開 Email？
- 是否只希望客戶透過 LINE 與電話聯絡？
- LINE ID 是否確認為 `chengyi-168`？
- 電話是否確認為 `0965-819-629`？
- 聯絡人是否確認為 `汪承漢`？

對應欄位：

- `{{EMAIL_PLACEHOLDER}}`
- NAP
- Footer
- LocalBusiness JSON-LD

### 11. 評價與客戶案例

- 是否可以提供 3-5 則真實客戶評價？
- 評價來源是 Google、Threads、Facebook 還是客戶私訊？
- 是否可以公開客戶稱呼？例如「高雄三民區林小姐」。
- 是否有搬家前後照片可以使用？
- 是否有特別成功的案例？例如夜間急件、無電梯公寓、精密設備搬運。

對應內容：

- Reviews 區塊
- Blog 客戶案例文章
- Local SEO 信任訊號

### 12. 品牌語氣與禁忌

- 客戶希望品牌看起來偏年輕、專業、親切，還是高級？
- 是否有不能使用的詞？例如「最便宜」、「保證最低價」。
- 是否能使用「Threads 網友推薦」這類說法？
- 是否有正式 slogan？
- 「承載所託 | 億心送達」是否確定使用？

對應內容：

- 首頁 Hero
- SEO title / description
- CTA 文案
- Blog 文章語氣

## 建議實際訪談順序

1. 先問地址、服務區域、營業時間，這三個最影響上線與 SEO。
2. 再問計價、夜間服務、包材規則，這些最影響客戶是否敢預約。
3. 接著問 Google Maps、社群、評價，這些最影響信任與 LocalBusiness schema。
4. 最後確認品牌語氣、服務主打項與不能寫的內容。
